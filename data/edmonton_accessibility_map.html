Edmonton Accessible Housing Map

A map to help find accessible housing in Edmonton — homes whose building permits mention ramps, lifts, wheelchair access, or barrier-free features. Click any dot for details and a Street View photo.
Why this exists & data source
The problem: there is no central list of which Edmonton homes are wheelchair-accessible or barrier-free, so finding accessible housing often means checking listings one at a time, with no way to search for the features that matter. The City's public building & development permits do record this work — ramps, lifts, barrier-free bathrooms — but it is buried in large datasets not built for this purpose.

How this helps: this tool pulls those accessibility-related permits (2009–present), narrows them to homes, and maps them so they can be browsed and filtered — a starting point for Spinal Cord Injury Alberta and the people it serves to find and track accessible housing.

It is an automatically generated draft, so some entries may be false matches (for example, a parking-garage “ramp”) — check the permit description in each popup. Not an official listing.
